# Giter8 Scala App Template

Giter8 Scala app template with Cats, FS2 and Weaver and more

### How to try

```shell
sbt new SiberianPathfinder/scala-app.g8
```

### Features

Dependencies:
- `Cats`: `core`, `effect`, `mtl`
- `FS2`: `core`, `io`
- `log4cats (slf4j)` 
- `weaver`

Plugins:
- `scalafix` + rules
- `scalafmt` + rules
- `assembly`
